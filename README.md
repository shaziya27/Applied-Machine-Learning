## Projects

### Task 1 – Customer Churn Classification

Developed a supervised machine learning model to predict customer churn using the Telco Customer Churn dataset. The project includes data preprocessing, exploratory data analysis, feature engineering, model training, and performance evaluation using multiple classification metrics.

**Highlights**
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Logistic Regression and Random Forest models
- Model evaluation using Accuracy, Precision, Recall, F1-Score, and ROC-AUC
- Model serialization for deployment

---

### Task 3 – Machine Learning Model Deployment

Deployed the trained customer churn prediction model as a REST API using FastAPI and containerized the application using Docker.

**Highlights**
- FastAPI REST API
- Model loading using Pickle
- Prediction endpoint
- Interactive Swagger documentation
- Docker containerization

---

### Task 4 – Responsible AI

Implemented Responsible AI techniques to improve model transparency and evaluate fairness using explainability tools and bias analysis.

**Highlights**
- SHAP explainability
- Feature importance analysis
- Fairness evaluation
- Bias identification
- Bias mitigation recommendations
